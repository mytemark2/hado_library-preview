簡易使い方説明
1. 上部のデータバーを確認
画面一番上の表示で、全データ/保存データ、保存名、武将状態、装備状態を確認できます。設定変更やJSON再読込は、この表示欄をタップして開く「データ管理」から行います。
2. JSONと保存データを準備
PCはJSONフォルダ、スマホはJSONファイルを読み込みます。既存の保存データがある場合はImport、新規で使う場合は保存データを作成してから所有武将・装備を登録します。
3. 目的に合う検索を選択
通常検索はキーワードとタグ、状態変化検索は状態変化の単一選択、型検索は攻撃速度型・撃心型などのプリセットと複数条件で候補を探します。タグは同じグループ内がOR、異なるグループ間がANDです。
4. 詳細・コピー・お気に入りを活用
検索結果から内容詳細、関連リンク、パラメータ、コピー用テキストを確認できます。武将・装備の★は、現在の保存データへ登録されます。
5. 部隊編成と軍馬編成へ進む
部隊編成では配置・侍従・参軍・兵器・武装・軍馬を設定し、変化率タブで状態変化率を確認します。軍馬編成では所有軍馬の作成・編集を行います。部隊への割当は部隊編成画面の軍馬枠で行います。
